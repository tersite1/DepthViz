## Overview

DepthViz is an iOS application that transforms your iPhone into a professional-grade point cloud scanner by integrating **LiDAR sensor**, **camera**, and **IMU sensor** data. Designed for iPhone 12 Pro and newer models, DepthViz is the first LiDAR scanner application developed in Korea, offering robust 3D data capture and management capabilities.

## üõ†Ô∏è Build Instructions

**Requirements:**
- iPhone 12 Pro or newer with LiDAR sensor
- Xcode installed on Mac
- USB cable for device connection

**Steps:**

1. **Clone the repository:**


2. **Open in Xcode:**
- Launch `DepthViz.xcodeproj`

3. **Connect your iPhone:**
- Connect via USB cable and select as build target

4. **Build and Run:**
- Click the "Run" button in Xcode

**Note:** For local builds, use the `DepthViz.zip` file and extract all contents to the project directory (excluding `DepthViz.xcodeproj`).
