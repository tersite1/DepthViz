DepthViz is an iOS application designed to function as a point cloud scanner by integrating data from the **LiDAR sensor**, **camera**, and **IMU sensor**.  
This app is specifically tailored to operate on iPhone 14 Pro and newer models equipped with a LiDAR sensor, offering a robust tool for capturing and managing 3D data.

As far as I know, DepthViz is the first LiDAR Scanner on iPhone in Korea.

### Accessibility
With the introduction of LiDAR on iPhone 14 Pro, Apple has made advanced 3D sensing technology accessible to everyone.  
Developers can now easily process LiDAR data and explore its full potential for creative and industrial applications.

<img width="1052" alt="lidar_example" src="https://github.com/tersite1/tersite1/assets/160453556/a3d51ee4-f032-4065-95e9-f1abbd1b34b1">

### Point Cloud Scanning
- **Advanced Scanning:** Capture detailed 3D point clouds of objects and environments.  
- **High Accuracy:** Fuses LiDAR, RGB, and IMU data for precise spatial reconstruction.

### File Management
- **Project-based Organization:** Manage scans under dedicated projects.  
- **Share & Delete:** Easily share or delete point cloud files directly from the app.

### Future Enhancements
- Integrated **PLY File Viewer** for in-app 3D visualization  
- Cloud sync for storing and sharing scans across devices
